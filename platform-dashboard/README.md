# platform-dashboard

Static carrier UI assets (HTML/CSS/JS). **Not** React/Next in this repo — vanilla assets only.

## Run locally

```bash
cd platform-dashboard/public
python3 -m http.server 3020
```

Open `http://127.0.0.1:3020/`.

For a production React/Next app, add a separate app under this folder without coupling to `platform-api`.
